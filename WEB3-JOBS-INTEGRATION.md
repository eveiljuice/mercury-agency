# Web3.Career API Integration

**TL;DR:** Интегрирован API Web3.career для отображения актуальных вакансий в Web3 и блокчейне. Добавлена новая секция с фильтрацией, красивым дизайном и правильными ссылками на web3.career (с rel="follow").

## Что сделано:

1. **Создан сервис для работы с API** (`src/services/web3Jobs.ts`)
   - Функции для получения вакансий с фильтрами (remote, по тегам, по стране)
   - TypeScript интерфейсы для типизации данных
   - Обработка ошибок API

2. **Создан компонент Web3Jobs** (`src/components/Web3Jobs.tsx`)
   - Красивый grid с карточками вакансий
   - Фильтрация: Все вакансии / Remote
   - Отображение: компания, дата, локация, теги
   - Правильные ссылки на web3.career с `rel="follow"` (без utm параметров)
   - Адаптивная верстка для мобильных устройств

3. **Добавлена мультиязычность** 
   - Переводы для RU/EN в секции Jobs
   - Все тексты интерфейса переведены

4. **Обновлена навигация**
   - Добавлен пункт "Web3 Jobs" в десктоп навигацию
   - Добавлен пункт в мобильное меню

5. **Обновлен .env.example**
   - Добавлен токен Web3 Career API

## Как проверить:

1. Запустить проект: `npm run dev`
2. Открыть в браузере
3. Кликнуть на "Web3 Jobs" в навигации или проскроллить до секции
4. Проверить фильтры: "All Jobs" / "Remote"
5. Убедиться что вакансии загружаются и отображаются корректно
6. Проверить переключение языков (RU/EN)
7. Кликнуть "Apply Now" на любой вакансии — должна открыться страница на web3.career

## Важно:

- Все ссылки на вакансии имеют `rel="follow"` как требует API
- Нет utm параметров в apply_url
- API токен: `kMn9WFPS2fhG2B38F9mqMh11VgepjAg7`
- Footer ссылается на web3.career с правильным атрибутом
